DBeaver连接MySQL 8以上版本以及解决可能遇到的问题

  • 前期准备
  • DBeaver连接MySQL
  • 解决方案

前期准备

数据库MySQL版本8.0.18

下载一个DBeaver.exe

下载地址：https://dbeaver.io/download/

DBeaver连接MySQL

点击新建 –》数据库连接 –》下一步

               

选择数据库 –》下一步 –》完成

                            

双击新建的文件会提示下面这样的错误

解决方案

原因是数据库的时区配置不对 打开cmd以管理员运行 登录数据库 –》输入 show variables like ‘%time_zone%';

SYSTEM为默认美国时间，解决方法：输入 set global time_zone='+8:00’;

虽然显示还是SYSTEM，退出数据库重启一遍即可

然后返回到DBeaver再次双击连接

红色框表示连接成功
